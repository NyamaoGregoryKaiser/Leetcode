# Sorting Algorithms - Documentation

This directory contains comprehensive documentation for the sorting algorithms and related problems implemented in this project. It's designed to provide in-depth understanding, conceptual clarity, and practical tips for coding interviews.

## Contents

*   **`algorithms-explained.md`**: Detailed explanations of how each sorting algorithm works, including their mechanics, properties (stability, in-place), and typical use cases.
*   **`visual-diagrams.md`**: ASCII art diagrams to visually represent the steps and concepts behind key sorting algorithms and partitioning techniques.
*   **`edge-cases-gotchas.md`**: A discussion of common edge cases, potential pitfalls, and important considerations when implementing or analyzing sorting algorithms.
*   **`interview-tips.md`**: Advice on how to approach sorting-related interview questions, what to discuss, common follow-up questions, and variations.

## How to Use

Feel free to read through these documents to deepen your understanding. They complement the code implementations by providing the theoretical background and practical wisdom necessary for coding interviews.

Each document is structured to be self-contained but also references other parts of the project where relevant.
