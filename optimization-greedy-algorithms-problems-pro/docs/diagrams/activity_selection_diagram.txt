**Legend:**
*   `[---]` = An activity
*   `_` = Free time slot
*   `X` = Selected activity
*   `>` = Last selected activity's finish time

This diagram illustrates how at each step, the activity with the earliest finish time among compatible options is chosen, maximizing the remaining available time for subsequent selections.
