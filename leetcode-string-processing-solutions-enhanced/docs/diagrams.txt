# Visual Diagrams for String Algorithms

## 1. Longest Palindromic Substring: Expand Around Center

**Concept:** A palindrome expands symmetrically from its center. The center can be a single character (odd length) or two characters (even length).

**Odd Length Palindrome (e.g., "racecar"):**

Input: `s = "racecar"`

Iteration `i` at 'e' (index 3):
Center: `e` (`left = 3`, `right = 3`)

1. Initial:
   ```
   r a c e c a r
         ^
       l,r
   ```

2. Expand 1: `s[2] == s[4]` ('c' == 'c')
   ```
   r a c e c a r
       ^   ^
       l   r
   ```

3. Expand 2: `s[1] == s[5]` ('a' == 'a')
   ```
   r a c e c a r
     ^       ^
     l       r
   ```

4. Expand 3: `s[0] == s[6]` ('r' == 'r')
   ```
   r a c e c a r
   ^           ^
   l           r
   ```

5. Expand 4: `left = -1`, `right = 7`. Boundary check fails.
   Length calculation: `right - left - 1 = 7 - (-1) - 1 = 7`.
   Palindrome: "racecar"

**Even Length Palindrome (e.g., "abba"):**

Input: `s = "abba"`

Iteration `i` at 'b' (index 1):
Center: `bb` (`left = 1`, `right = 2`)

1. Initial:
   ```
   a b b a
     ^ ^
     l r
   ```

2. Expand 1: `s[0] == s[3]` ('a' == 'a')
   ```
   a b b a
   ^     ^
   l     r
   ```

3. Expand 2: `left = -1`, `right = 4`. Boundary check fails.
   Length calculation: `right - left - 1 = 4 - (-1) - 1 = 4`.
   Palindrome: "abba"
